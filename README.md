# Лабораторна робота №7

## Завдання 1

### Короткий опис задачі
Програма зчитує числа з файлу `number.txt` та знаходить добуток усіх чисел, які кратні 4.

### Опис алгоритму розв’язання
1. Відкрити файл `number.txt`.
2. Зчитувати числа по одному.
3. Перевіряти, чи число кратне 4.
4. Якщо так — перемножити його до результату.
5. Вивести добуток або повідомлення, якщо таких чисел немає.

### Пояснення роботи програми
- Дані зчитуються з файлу у циклі.
- Кожне число перевіряється: `number % 4 == 0`.
- Якщо умова виконується — число множиться до змінної добутку.
- Після завершення виводиться результат.

## Завдання 2

### Короткий опис задачі
Програма зчитує матрицю 5×5 з файлу `2D.txt` та знаходить мінімальний елемент головної діагоналі.

### Опис алгоритму розв’язання
1. Відкрити файл `2D.txt`.
2. Зчитати матрицю 5×5.
3. Вивести матрицю.
4. Знайти мінімальний елемент серед `matrix[i][i]`.
5. Вивести результат.

### Пояснення роботи програми
- Матриця зчитується у двовимірний масив.
- Головна діагональ проходить через елементи з однаковими індексами.
- Виконується порівняння елементів для пошуку мінімального.
